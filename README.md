# Muhammad Rizki Fani — CV

This is a Vercel-ready static site containing Muhammad Rizki Fani's CV.

Quick deploy

- Option A — Vercel (recommended):
  1. Install Vercel CLI: `npm i -g vercel`
  2. From this folder run: `vercel`
  3. Follow prompts to deploy (or run `vercel --prod` for production)

- Option B — GitHub integration:
  1. Push this repository to GitHub.
  2. Import the repo in Vercel dashboard and deploy.

Local dev

- Run `npm run start` (requires Vercel CLI)

Files

- index.html — the CV page
- vercel.json — Vercel config for static deployment
